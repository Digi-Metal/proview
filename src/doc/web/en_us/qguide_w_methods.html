Web Methods


Methods are functions that act on an object. The methods are activated from the popup menu in most places where the object is displayed: process graphics, the navigator, plc trace, alarm and event lists.


These are the web methods of an ordinary object:

Graph Open process graphics for the object
Help Display a help text the object
Navigator Display the object in navigator
Open Plc Open the plc code with trace
Crossreferences References of the object in the plc code
DataSheet Open a data sheet for the object
ClassHelp Open a helptext for the class of the object
